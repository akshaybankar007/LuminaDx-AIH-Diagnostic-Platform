# LuminaDx AIH Diagnostic Platform

LuminaDx is a clinical decision support platform that automates Autoimmune Hepatitis (AIH) scoring using IAIHG revised criteria. It processes serological and histological data via an AI engine to generate classified diagnostic reports and clinical narratives.

## Key Features
* **AI-Assisted Scoring**: Automates IAIHG calculations based on patient intake data.
* **Responsive Interface**: Clinical dashboard optimized for desktop and mobile devices.
* **Clinical Documentation**: Generates professional, print-ready PDF diagnostic reports.
* **Secure Sessions**: Localized session management for patient data handling.

## Technical Stack
* **Frontend**: HTML5, CSS3 (Custom Responsive Framework), JavaScript (ES6+).
* **Backend**: Node.js, Express.
* **AI Model**: Google Gemini 2.5 Flash.
* **PDF Engine**: PDFKit.
